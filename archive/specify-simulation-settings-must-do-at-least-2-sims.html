## Static netdx
Issue [#134](https://github.com/statnet/SHAMP/issues/134) summarizes some problems encountered and corresponding solutions in the first post. The TOC links to in-depth investigations of the internals of static and dynamic netdx calls. Two important takeaways:

1. The default starting net for netdx simulations is the empty network
2. Using sequential=TRUE is like running a dynamic sim in that each sim depends on the previous one, sort of like a time step. However there is no dissolution.



See ```[SHAMP/Epidemics/netdx_diagnostics/current/netdx_static_b1i1e6_pers.html]```

## Dynamic netdx, homogeneous dissolution
Issue [#134](https://github.com/statnet/SHAMP/issues/134) summarizes some problems encountered and corresponding solutions in the first post. The TOC links to in-depth investigations of the internals of static and dynamic netdx calls. There were two important takeways:

1. The default starting net for the netdx simulations was the empty network. We used ee.netest to use the fitted net instead.
2. The MCMC.burnin parameter essentially controls the MCMC interval for dynamic sims, since it is a "burnin per time step."

[This post in Issue #134](https://github.com/statnet/SHAMP/issues/134#issuecomment-413910396) gives the overview of netdx vs ee.netdx - we used the latter function to achieve #1.

Issue [#146](https://github.com/statnet/SHAMP/issues/146) summarizes both homogeneous and heterogeneous dissolution diagnostics pretty thoroughly. For homogeneous dissolution, diagonstics look good except for nodematched terms in the cohab model, which plateau too low. This is noted in [this #134 post](https://github.com/statnet/SHAMP/issues/134#issuecomment-417759234) as well.

See ```[SHAMP/Epidemics/netdx_diagnostics/current/netdx_dynamic_pers_MCMC.burnin_setbasis.html]```

## Dynamic netdx, nodefactor(race) dissolution
Issue [#129](https://github.com/statnet/SHAMP/issues/129) goes over how we calculated the target stats, resolving questions regarding weights and the exponential mean implied by the median.

Issue [#146](https://github.com/statnet/SHAMP/issues/146) summarizes both homogeneous and heterogeneous dissolution diagnostics pretty thoroughly. For both networks, the percent errors are higher for some terms than in the homogeneous dissolution diagnostics. We hit the cohab durations by race pretty well except for Hispanics - too high. Pers durations are good for H and HI but too high for B and BI, and too high overall.
