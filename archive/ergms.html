7.2 History

  • The following chapters describe our final models to date, which were fitted using the scripts in SHAMP/egonet/analyses/ergms_final
  • For historical models see: SHAMP/egonet/analyses/ergms/model-fits and SHAMP/egonet/analyses/ergms/archive
  • Note that the models in SHAMP/egonet/analyses/ergms_final are repeated fits of the models of the same name in SHAMP/egonet/analyses/model-fits, but using the Feb 28 2019 version of ergm.ego-private, branch dev and the April 10 2019 version of ergm-private branch dev. The one other difference is that for the cohab model, the MCMC interval was increased from 1e5 to 1e6.
